Ticketing — Web app per eventi e biglietteria online

Piattaforma per navigare eventi, scegliere posti e acquistare biglietti in sicurezza. Front-end in HTML/CSS, back-end in PHP, database MySQL gestito via phpMyAdmin.

HTMLCSSPHPMySQLphpMyAdmin
Funzionalità Architettura Database & Query
WCAG AA Responsive

Funzionalità principali

Catalogo & Ricerca

Lista eventi con filtri per città, data, categoria e fascia prezzo.

  • Card evento con posti disponibili
  • Utilizzo barra di ricerca

Selezione quantità di biglietti e acquisto

Possibilità di aggiungere il numero di biglietti acquistati

  • Acquisto solo se effettuato il login

Profilo utente

Accesso ad informazioni personali e Storico ordini

  • Modifica dati e password

Amministrazione

Accesso e possibilità di modifica o aggiunta di eventi

  • Gestione prezzi e sconti

Accessibilità

Scelte di UI per leggibilità e inclusione.

  • Contrasto e gerarchie tipografiche
  • Alt testuali e focus visibile
  • Obiettivo AA (WCAG 2.0)

Architettura tecnica

Front-end

  • HTML semantico + CSS modulare
  • Form accessibili con label/aria-*

Back-end & Sicurezza

  • PHP con sessioni e ruoli (user/admin)
  • Prepared statements per query

Schema Database (principali tabelle)

UTENTI

  • email, username (PK), password, acquisti(NULL)
  • nome, cognome
  • Query: SELECT/INSERT/UPDATE password

BIGLIETTI

  • id(PK), nome evento, data e ora, luogo, categoria, band (NULL), descrizione, quantità di biglietti
  • fasce orarie (slot)
  • Query: SELECT/UPDATE quantita

AMMINISTRATORI

  • username, email (PK), password
  • Accesso alle funzioni di gestione (aggiungere/togliere/modificare eventi)

Sono inclusi flussi di convalida, messaggi d’errore, e aggiornamento stato slot in fase di acquisto.
