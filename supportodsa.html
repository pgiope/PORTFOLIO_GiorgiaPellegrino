SupportoDSA — Web app per tutoraggio e prenotazioni

Piattaforma per famiglie e tutor dedicata al supporto compiti per ragazzi con DSA. Front-end in HTML/CSS, back-end in PHP, e database MySQL gestito via phpMyAdmin.

HTMLCSSPHPMySQLphpMyAdmin
Funzionalità Architettura Database Documento
AA WCAG target Locale dev

Obiettivo e target

Semplificare l’accesso al servizio di tutoraggio: gli utenti possono registrarsi, prenotare ore con i tutor disponibili, e gestire il proprio profilo; l’amministratore aggiorna tutor e serate informative.

Funzionalità principali

Registrazione & Login

Validazione form, gestione errori, feedback all’utente.

  • Registrazione nuovo utente
  • Login utente/admin/tutor
  • Reset/modifica password

Prenotazione lezioni

Selezione tutor, giorno e fascia oraria.

  • Controllo autenticazione
  • Verifica disponibilità e conferma
  • Blocco slot prenotati

Area personale

Vista differenziata per ruoli.

  • Utente: lezioni prenotate
  • Tutor: disponibilità settimanali
  • Entrambi: cambio password

Amministrazione

Pannello dedicato.

  • Gestione serate informative
  • Inserimento/Cancellazione tutor
  • Controlli su esistenza dati

Accessibilità

Scelte di UI per leggibilità e inclusione.

  • Contrasto e gerarchie tipografiche
  • Alt testuali e focus visibile
  • Obiettivo AA (WCAG 2.0)

Architettura tecnica

Front-end

HTML semantico + CSS modulare.

  • Layout responsive e componenti riutilizzabili
  • Form con etichette/aria-label e messaggi di errore

Back-end & DB

PHP con MySQL via phpMyAdmin.

  • Autenticazione per ruoli (utente, tutor, admin)
  • Operazioni CRUD su utenti, tutor, disponibilità, serate
  • Query di controllo esistenza/duplicati e aggiornamenti

Schema Database (principali tabelle)

UTENTI

  • email, username (PK), password
  • nome, cognome
  • Query: SELECT/INSERT/UPDATE password

TUTOR

  • nome, cognome, email (PK), età
  • descrizione
  • Query: SELECT su email, elenco tutor

DISPONIBILITÀ

  • tutor (FK)
  • fasce orarie (slot)
  • Query: SELECT per tutor

SERATE_INFORMATIVE

  • tema, data, descrizione
  • Gestite da admin

AMMINISTRATORI

  • nome, cognome, email (PK), password
  • Accesso alle funzioni di gestione

SERVIZI

  • Catalogo/descrizioni
  • Uso informativo sul sito

Sono inclusi flussi di convalida, messaggi d’errore, e aggiornamento stato slot in fase di prenotazione.
